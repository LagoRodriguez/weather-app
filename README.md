# Weather App

## Instrucciones


English version

	Description
	Ayesa Weather App is a web application that allows users to search and view weather information for different cities. It also includes additional features such as saving favorite cities and switching between Celsius and Fahrenheit.

	Requirements
	Before you get started, ensure you have the following installed:

	Node.js (Recommended version: 20.x or higher)
	A code editor like Visual Studio Code

	Installation
	Clone the Repository:

	git clone https://github.com/lagorodriguez/ayesaweatherapp.git
	cd ayesaweatherapp


	Install Dependencies:
	npm install
   
	Set up .env.local File:
	Create a .env.local file at the root of your project with the following content:
	VITE_API_KEY="your_weatherapi_key"

	Running the Application
	npm run dev

	Features
	1. Search Weather by City
	Enter a city in the search field.
	Click the "Search" button or press Enter to get weather information.
	2. Detailed Weather Display
	Once a city is found, detailed weather information such as current temperature in Celsius and Fahrenheit, weather description, and an icon representing the weather will be displayed.
	3. Save Favorite Cities
	After searching for a city, you can save it to your favorite cities list.
	4. Unit Select
	Click the "Select unit" select to between temperature units.
	5. Error Handling
	If an error occurs while searching for a city, an error message will be displayed on the screen.
	Customization
	You can modify the API routes in the App.jsx file to adapt to other weather data sources if needed.
	Contributing
	Contributions are welcome! If you have any improvements or features to add, please create a pull request.
